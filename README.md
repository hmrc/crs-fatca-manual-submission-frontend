
# crs-fatca-manual-submission-frontend

This service is used to manual file upload process.

## Running the service

Service manager: CRS_FATCA_ALL

Port: 10039

Link: http://localhost:10039/crs-fatca-manual-submission-frontend

 ---

## Running Tests
Run unit tests:
 ```
 sbt test
 ```

Run integration tests:
 ```
 sbt it:test
 ```
